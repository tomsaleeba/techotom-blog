Steps:

  1. Install X11, lirc and some other needed dependencies (including alsa)
  2. Install video card drivers (nvidia in my case) (don't forget make gcc linux-headers)
  3. Add the xmbc repo
  4. install xbmc
  5. allow your xbmc user to access your audio device (can check with aplay -l)
  6. make xbmc startup automatically
