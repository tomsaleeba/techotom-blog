  1. install tomcat with apt-get
  2. download and deploy Jenkins
  3. use usermod -s /bin/bash to give the tomcat7 user a shell
  4. su -c "/bin/bash" tomcat7 to start a shell as tomcat7
  5. ssh-keygen to generate a key (be sure to use the default id_rsa name and have no passphrase)
  6. cat the .pub key and chuck it onto BitBucket
  7. ssh-add the key
  8. might need to start ssh-agent
  9. ssh git@bitbucket.org to add the key to the known hosts and test it out
