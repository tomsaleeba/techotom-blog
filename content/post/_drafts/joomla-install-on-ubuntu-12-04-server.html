1. sudo apt-get install php5 mysql-server php5-mysqlnd
2. download Joomla
3. unzip to /var/www/joomla
4. touch configuration.php
5. sudo chmod -R 777 /var/www/joomla
6. open your browser to hostname/joomla/index.php
